# Contributing to Muslim Daily

Terima kasih atas minat Anda untuk berkontribusi pada Muslim Daily! 🎉

## Cara Berkontribusi

### 1. Fork Repository
Fork repository ini ke akun GitHub Anda.

### 2. Clone Repository
```bash
git clone https://github.com/username/muslim-daily.git
cd muslim-daily
```

### 3. Install Dependencies
```bash
npm install
```

### 4. Buat Branch Baru
```bash
git checkout -b feature/nama-fitur-anda
```

### 5. Lakukan Perubahan
- Pastikan kode Anda mengikuti style guide yang ada
- Tulis kode yang clean dan mudah dipahami
- Tambahkan komentar jika diperlukan

### 6. Test Perubahan
```bash
npm run dev
```
Pastikan aplikasi berjalan dengan baik dan tidak ada error.

### 7. Commit Perubahan
```bash
git add .
git commit -m "feat: deskripsi singkat perubahan"
```

Format commit message:
- `feat:` untuk fitur baru
- `fix:` untuk bug fix
- `docs:` untuk perubahan dokumentasi
- `style:` untuk perubahan styling
- `refactor:` untuk refactoring kode

### 8. Push ke GitHub
```bash
git push origin feature/nama-fitur-anda
```

### 9. Buat Pull Request
Buat Pull Request dari branch Anda ke branch `main` repository utama.

## Panduan Kode

### Style Guide
- Gunakan TypeScript untuk type safety
- Ikuti konvensi penamaan yang ada
- Gunakan Tailwind CSS untuk styling
- Komponen harus responsive dan mobile-friendly

### Struktur Folder
```
muslim-daily/
├── app/              # Next.js app router
├── components/       # React components
├── public/          # Static assets
└── README.md        # Dokumentasi
```

### Best Practices
- Komponen harus reusable
- Hindari hardcoded values
- Gunakan TypeScript interfaces
- Optimasi performa
- Accessibility compliant
